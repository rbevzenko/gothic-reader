# Gothic Reader

**Инструмент для работы с историческими немецкими текстами в готическом шрифте (Fraktur).**

Загружайте PDF-книги XIX века — приложение автоматически преобразует готический шрифт в современный немецкий или переводит на другой язык. Работает прямо в браузере, без установки.

🔗 **[Открыть приложение](https://rbevzenko.github.io/gothic-reader/)**

---

## Что умеет

### Основной режим — Fraktur → Modernes Deutsch
Загрузите PDF, выберите страницы и запустите обработку. Приложение читает готический шрифт и выдаёт современный немецкий текст с исправленной орфографией (ſ→s, устаревшие написания, регистр).

### Перевод на другой язык
Переключите режим — и тот же PDF будет переведён на русский, английский, французский или польский.

### Возможности
- **Пакетная обработка** — весь PDF, заданный интервал (стр. 4–17) или список страниц (1, 3, 5–8, 12)
- **Просмотр постранично** — миниатюры всех страниц, навигация стрелками
- **Экспорт в PDF** — сохранить всё или только текущую страницу
- **Копирование** текста в буфер обмена
- **Одиночный режим** — URL из библиотеки dlc.mpg.de или загрузка скриншота
- **Счётчик времени** — показывает, сколько осталось до завершения обработки
- **Фоновая обработка** — вкладку можно свернуть, скорость не снижается

## Как пользоваться

### 1. Получить API-ключ Anthropic

Приложение использует модели Claude для распознавания и перевода. Ключ нужно получить один раз на [console.anthropic.com](https://console.anthropic.com/).

> **Стоимость:** ~$0.001–0.005 за страницу в режиме Haiku, ~$0.01–0.05 в режиме Sonnet.

### 2. Вставить ключ в приложение

Ключ сохраняется в `localStorage` браузера — вводить повторно не нужно.

### 3. Загрузить PDF

Перетащите файл или нажмите на зону загрузки. Миниатюры всех страниц появятся в полосе.

### 4. Выбрать режим, модель и страницы

| Параметр | Варианты |
|----------|----------|
| **Вывод** | Modernes Deutsch / Перевод (RU, EN, FR, PL) |
| **Модель** | Haiku 4.5 · Sonnet 4.5 · Opus 4 |
| **Страницы** | Все / Интервал (от–до) / Список (1, 3, 5–8) |

### 5. Запустить обработку

Нажмите **▶ Обработать все страницы**. Прогресс и оставшееся время отображаются в реальном времени.

### 6. Сохранить результат

- **↓ PDF** в шапке правой панели — текущая страница
- **Сохранить результат как PDF** — все обработанные страницы одним документом
- **Kopieren** — скопировать текст в буфер

## Выбор модели

По умолчанию приложение выбирает модель автоматически в зависимости от режима.

| Модель | Стоимость | Режим по умолчанию |
|--------|-----------|--------------------|
| **Haiku 4.5** | ~$0.25 / 1M токенов | Fraktur → Modernes Deutsch |
| **Sonnet 4.5** | ~$3 / 1M токенов | Перевод на другой язык |
| **Opus 4** | ~$15 / 1M токенов | Максимальное качество |

## Технические детали

- **Чистый браузерный JavaScript** — никакого бэкенда, никакой установки
- **[PDF.js](https://mozilla.github.io/pdf.js/)** — разбивка PDF на страницы и рендеринг
- **OffscreenCanvas** — рендеринг страниц работает в фоновых вкладках без замедления
- **[Anthropic Claude API](https://www.anthropic.com/)** — прямые запросы из браузера
- Опциональный **screenshot-server** (Node.js + Playwright) для обхода защиты на dlc.mpg.de

## Локальный запуск

Достаточно открыть `index.html` в браузере — никаких зависимостей и сборки не требуется.

```bash
git clone https://github.com/rbevzenko/gothic-reader.git
cd gothic-reader
open index.html        # macOS
# или просто перетащите файл в браузер
```

### Опциональный screenshot-сервер (для dlc.mpg.de)

```bash
cd screenshot-server
npm install
npm start
# Сервер запустится на http://localhost:3001
```

## Ограничения

- Работает лучше всего с PDF хорошего качества (300+ DPI)
- Рукописные тексты распознаются хуже типографских
- Для больших PDF (100+ страниц) рекомендуется обрабатывать частями
- API-ключ хранится в `localStorage` — не используйте на чужих компьютерах
